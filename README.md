# Real-Time Dashboard UI

React + TypeScript + Vite frontend for the real-time market price dashboard.

## Setup

```bash
npm install
npm run dev
```

Runs at `http://localhost:5173`. Requires the API server running at `http://localhost:3000`.

## Stack

- React 19, TypeScript, Vite
- Tailwind CSS v4
- WebSocket for live price updates
